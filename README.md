# Simple Calculator in Python

A simple Python program that allows the user to perform basic arithmetic operations: **addition, subtraction, multiplication, and division**.

This program is perfect for beginners who are learning Python and want to practice **user input**, **conditional statements**, and **basic arithmetic operations**.

---

## Features

- Perform **addition (+)**, **subtraction (-)**, **multiplication (*)**, and **division (/)**.
- Accepts **two numbers** as input from the user.
- Displays the result of the selected operation in a **clear format**.

---

## How It Works

1. The program prompts the user to **enter two numbers**.
2. The program prompts the user to **choose a mathematical operation** (`+`, `-`, `*`, `/`).
